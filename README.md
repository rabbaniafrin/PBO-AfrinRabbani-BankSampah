# Sistem Informasi Bank Sampah (SIBAS)

Project tugas PBO - Phase 1

## Deskripsi
Sistem ini dibuat untuk membantu pengelolaan bank sampah secara digital. 
Nasabah dapat menyetor sampah dan sistem akan menghitung nilai ekonomi dari sampah tersebut.

## Entitas Sistem
1. Nasabah
2. Sampah
3. Transaksi
